# Jeu des Drapeaux
## Projet réaliser par Adrien TRAVAILLE

### Mode de jeu
 Le jeu comporte 6 modes de jeu 
  - Monde complet
  - Europe
  - Asie
  - Afrique
  - Amérique
  - Océanie

## Victoire
Chaque fois que le jouer trouve une bonne réponse, le compteur de score s'incrémente de 1
